# Privacy Policy

Smart Playlist Player does not collect or transmit any personal data.

## Data Usage
- Reads local audio metadata via MediaStore for playback.
- Stores play history and favorites locally on device using Room.
- No analytics, no ads, no network requests.

## Permissions
- READ_MEDIA_AUDIO / READ_EXTERNAL_STORAGE: access local audio files.
- POST_NOTIFICATIONS: show playback controls.
- FOREGROUND_SERVICE / FOREGROUND_SERVICE_MEDIA_PLAYBACK: keep audio running in background.

## Contact
If you have questions about privacy, please open an issue in the repository.
