# Contributing

Thanks for considering contributing to Smart Playlist Player.

## How to contribute
1. Fork the repository and create a feature branch.
2. Keep changes focused and small.
3. Run the app and verify playback, notifications, and scanning still work.
4. Update README/CHANGELOG if you add user-facing features.
5. Open a pull request with a clear description and screenshots when relevant.

## Code style
- Kotlin style: follow the official Kotlin style guide.
- Keep UI changes minimal and consistent with Material3.
- Avoid introducing network access or analytics without discussion.

## Issue reporting
Please include device model, Android version, and steps to reproduce.
