# Store Listing Draft

## App Name
Smart Playlist Player

## Short Description
Local-first music player with smart queues, offline scan, and background playback.

## Full Description
Smart Playlist Player is an offline-focused Android music player that scans your local library, builds smart queues
from listening history, and keeps playback stable in the background. It combines a clean Compose UI with Media3
MediaSession integration, so notification, lock screen, and headset controls stay in sync.

Highlights:
- Fast local scan via MediaStore
- Smart queue generation based on favorites and play history
- Full Now Playing screen with seek and queue preview
- Background playback with notification controls
- No network access; data stays on device

## Key Features
- Local scan and instant playback
- Favorites and intelligent queues
- Background playback with media controls
- Now Playing with seek and queue preview

## Keywords
music player, offline, smart playlist, local library, media session, exoplayer
