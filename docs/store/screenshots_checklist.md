# Screenshots Checklist

## Required Screens
- Home screen (library + scan + now playing entry)
- Now Playing screen (seek bar + controls + queue)
- Smart Queue screen (generated list)
- Notification controls
- Lock screen controls

## Suggested Resolution
- 1080x2400 (portrait)

## Capture Notes
- Use real device if possible.
- Ensure a real track is playing for Now Playing/notification screenshots.
- Avoid showing personal information in screenshots.
